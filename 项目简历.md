# 项目经历（简历版）

## 一句话版本（可放简历项目标题下）
独立设计并持续迭代本地化多智能体 RAG 系统（FastAPI + LangGraph + React），实现文档上传索引、混合检索、图谱检索、流式问答、权限隔离与运维治理能力，支持企业私有化部署场景下的稳定性与可审计要求。

## 项目名称
Multi-Agent Local RAG（企业内部知识问答与运维治理平台）

## 项目时间
2026.04 - 至今（已发布 v0.1.0 / v0.2 / v0.2.1 / v0.2.2）

## 技术栈
- 后端：Python 3.11、FastAPI、Pydantic、Uvicorn
- 智能体与检索：LangGraph、LangChain、ChromaDB、Neo4j、BM25、Reranker
- 模型与工具：OpenAI API / Ollama、Sentence-Transformers、DDGS、HTTPX、Tenacity
- 前端：React + TypeScript + Vite
- 工程化：Pytest、Docker Compose（Neo4j）、环境化配置（`.env`）

## 项目背景
面向私有化和本地部署知识问答场景，搭建兼顾效果、稳定性与安全治理的多智能体 RAG 平台，支持多轮会话、知识检索、权限隔离、审计追踪与管理端运营分析。

## 核心职责
1. 负责整体架构设计与核心模块开发（智能体编排、检索链路、API、权限与审计）。
2. 设计并实现文档处理与索引流程（PDF/图片 OCR、切分、增量入库、自动监听）。
3. 打通前后端全链路，落地会话管理、流式问答、后台治理和运维指标能力。
4. 建立回归测试体系，覆盖鉴权、权限、检索策略、治理接口与并发稳定性场景。

## 关键实现
1. 多智能体工作流与路由
- 基于 LangGraph 设计 Router、Vector RAG、Graph RAG、Web Research、Synthesis 协作流程。
- 引入查询分类与策略分流（如 `pdf_text` / `cybersecurity` / `general`），提升命中率与响应匹配度。

2. 混合检索与图谱增强
- 融合向量检索（Chroma）+ 关键词检索（BM25）+ 重排，优化召回质量。
- 接入 Neo4j 图谱检索，实现结构化关系信息补充答案。

3. 工程化文档索引
- 支持 PDF、图片 OCR、文本类文档统一入库。
- 实现自动增量索引监听，保证文件稳定后再入库，降低脏数据与重复写入。

4. 安全治理与后台能力
- 落地 JWT + RBAC + 用户级数据隔离与可见性控制（私有/公共）。
- 建设管理员治理接口（运营看板、趋势分析、导出能力、审计日志过滤）。
- 支持高权限操作保护（管理员创建审批令牌校验、密码重置、审批令牌重置）。

5. v0.2.2 重点升级（2026-04-09）
- 新增运行时韧性与治理模块：告警、后台任务队列、Bulkhead 隔离、Hybrid Executor、查询守卫、额度守卫、查询结果缓存。
- 新增运维脚本：混沌演练、负载/性能检查、数据迁移辅助脚本。
- 新增并发回归测试，提升高并发与降级场景下的稳定性保障。
- 更新工作流、流式链路、Neo4j 集成、API 与 Schema 契约，并强化 CI 质量门禁。

## 项目结果

- 形成可本地化部署、可权限隔离、可审计追踪的企业知识问答底座。
