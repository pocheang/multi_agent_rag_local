### 版本控制标准化

采用语义化版本控制（Semantic Versioning）:

```
v{MAJOR}.{MINOR}.{PATCH}

MAJOR: 重大架构变更、不兼容更新
MINOR: 新增功能、向后兼容
PATCH: Bug修复、文档更新
```
