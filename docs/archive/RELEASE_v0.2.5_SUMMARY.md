## 📋 发布概览

v0.2.5 是一个质量改进版本，主要基于深度代码审查修复了 18 个已识别问题，提升系统稳定性和性能。

---

## 🎯 发布目标

1. 修复多智能体 RAG 系统中的逻辑问题
2. 提升检索质量和路由准确性
3. 优化并发性能和超时控制
4. 增强系统稳定性和可靠性

---

## ✨ 主要改进

### 核心修复
- **18 个问题修复**: 涵盖 P0-P3 优先级
- **测试验证**: 29/29 tests passed
- **性能优化**: 降低延迟，减少 API 调用
- **质量提升**: 改进排序、引用和证据评分

### 技术亮点
- 检索策略参数传递修复
- Hybrid 路由并发逻辑修复
- Query Rewrite 去重和超时控制
- TTLCache 并发性能优化
- Graph Signal Score 优化

## ⚠️ 向后兼容性

**5 个行为变化需要注意**:
1. Query Rewrite 去重可能减少变体数量
2. Hybrid 路由不再并发执行（顺序执行）
3. Web Fallback 语义从"是否启用"改为"是否允许"
4. Reranker Fallback 分数归一化到 [0, 1]
5. Citation 句子分割使用 NLTK
