### 3. 用户功能

#### 会话管理
**功能**:
- ✅ 创建会话
- ✅ 列出会话
- ✅ 删除会话
- ✅ 会话历史

**API**:
```python
# 创建会话
POST /sessions
{
  "name": "My Session"
}

# 列出会话
GET /sessions

# 删除会话
DELETE /sessions/{session_id}
```

---

#### 文档上传
**功能**:
- ✅ 文件上传
- ✅ 文档索引
- ✅ 文档删除
- ✅ 文档列表

**支持格式**:
- PDF
- TXT, MD
- CSV, JSON
- 图片（PNG, JPG, etc.）

**API**:
```python
# 上传文档
POST /upload
{
  "file": <file>,
  "visibility": "private" | "public"
}

# 索引文档
POST /index/{doc_id}

# 删除文档
DELETE /documents/{doc_id}
```

---

#### 查询接口
**功能**:
- ✅ 流式查询
- ✅ 同步查询
- ✅ 会话上下文
- ✅ 源文档过滤

**API**:
```python
# 流式查询
POST /query
{
  "question": "What is RAG?",
  "session_id": "session123",
  "allowed_sources": ["doc1", "doc2"]
}

# 同步查询
POST /query-sync
{
  "question": "What is RAG?"
}
```

---

### 4. 用户隔离

#### 数据隔离
**功能**:
- ✅ 用户级数据隔离
- ✅ 会话隔离
- ✅ 文档隔离
- ✅ 查询隔离

**实现**:
- 所有数据关联用户 ID
- 查询时自动过滤用户数据
- 防止跨用户数据访问

---

#### 检索白名单
**功能**:
- ✅ 源文档白名单
- ✅ 用户级白名单
- ✅ 会话级白名单
- ✅ 查询级白名单

**使用**:
```python
# 限制检索源
POST /query
{
  "question": "...",
  "allowed_sources": ["doc1", "doc2"]
}
```

---

### 5. 管理功能

#### 文件可见性控制
**可见性级别**:
- **Private（私有）**: 仅上传者可见
- **Public（公开）**: 所有用户可见

**管理**:
```python
# 设置文件可见性
PUT /documents/{doc_id}/visibility
{
  "visibility": "public"
}
```

---

#### OCR 配置
**功能**:
- ✅ Tesseract OCR 支持
- ✅ 图片文本提取
- ✅ PDF 图片 OCR
- ✅ OCR 配置选项

**配置**:
```bash
# 环境变量
OCR_ENABLED=true
TESSERACT_PATH=/usr/bin/tesseract
```
