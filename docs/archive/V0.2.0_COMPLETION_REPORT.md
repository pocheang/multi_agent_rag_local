### 2. 用户管理功能

#### 管理员配置流程
**端点**: `POST /admin/users/create-admin`  
**功能**:
- ✅ 创建管理员账户
- ✅ 批准令牌验证
- ✅ 角色权限分配
- ✅ 审计日志记录

**配置流程**:
1. 提交管理员创建请求
2. 验证批准令牌
3. 创建管理员账户
4. 分配管理员权限
5. 记录审计日志

---

#### 用户分类
**四种用户类型**:
- **Admin（管理员）**: 完全权限
- **Power User（高级用户）**: 扩展权限
- **Standard User（标准用户）**: 基本权限
- **Guest（访客）**: 只读权限

**分类管理**:
```python
# 更新用户分类
PUT /admin/users/{user_id}/classification
{
  "classification": "power_user"
}
```

---

#### 安全操作
**密码重置**:
```python
POST /admin/users/{user_id}/reset-password
{
  "new_password": "secure_password"
}
```

**批准令牌重置**:
```python
POST /admin/users/{user_id}/reset-approval-token
```

**用户状态管理**:
```python
PUT /admin/users/{user_id}/status
{
  "status": "active" | "suspended" | "disabled"
}
```
