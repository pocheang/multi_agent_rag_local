## ⚠️ 重要变化

### 行为变化
1. Query Rewrite 去重可能减少变体数量
2. Hybrid 路由不再并发执行（顺序执行）
3. Web Fallback 语义从"是否启用"改为"是否允许"
4. Reranker Fallback 分数归一化到 [0, 1]
5. Citation 句子分割使用 NLTK
